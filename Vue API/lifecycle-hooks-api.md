# Vue Lifecycle Hooks API

> Источники: [Composition API: Lifecycle Hooks](https://vuejs.org/api/composition-api-lifecycle), [Options: Lifecycle](https://vuejs.org/api/options-lifecycle)
>
> **XLSX:** [`lifecycle-hooks-api.xlsx`](./lifecycle-hooks-api.xlsx) (генерация: `py -3 scripts/generate-lifecycle-hooks-api-xlsx.py`)

| API | Описание | По умолчанию | Пример | Детали | Ссылки |
|-----|----------|--------------|--------|--------|--------|
| `onBeforeMount()` | Регистрирует хук непосредственно перед первым монтированием компонента. | — | <pre><code>import { onBeforeMount } from 'vue'<br><br>onBeforeMount(() => {<br>  // reactive state готов, DOM ещё не создан<br>})</code></pre> | Composition API. Вызывать синхронно в setup(). Reactive state уже настроен, DOM-узлы ещё не созданы. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `onMounted()` | Регистрирует callback после монтирования компонента. | — | <pre><code>import { ref, onMounted } from 'vue'<br><br>const el = ref()<br>onMounted(() => {<br>  console.log(el.value)<br>})</code></pre> | Composition API. Компонент смонтирован: DOM создан и вставлен, синхронные дочерние компоненты смонтированы. Для side effects с доступом к DOM. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `onBeforeUpdate()` | Регистрирует хук перед обновлением DOM из-за изменения реактивного состояния. | — | <pre><code>import { onBeforeUpdate } from 'vue'<br><br>onBeforeUpdate(() => {<br>  // доступ к DOM до обновления Vue<br>})</code></pre> | Composition API. Можно безопасно изменять state. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `onUpdated()` | Регистрирует callback после обновления DOM из-за изменения реактивного состояния. | — | <pre><code>import { ref, onUpdated } from 'vue'<br><br>const count = ref(0)<br>onUpdated(() => {<br>  console.log(document.getElementById('count').textContent)<br>})</code></pre> | Composition API. Хук родителя вызывается после дочерних. Не мутировать state в updated — риск бесконечного цикла. Для конкретного изменения DOM используйте nextTick(). Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `onBeforeUnmount()` | Регистрирует хук непосредственно перед размонтированием экземпляра. | — | <pre><code>import { onBeforeUnmount } from 'vue'<br><br>onBeforeUnmount(() => {<br>  // экземпляр ещё полностью функционален<br>})</code></pre> | Composition API. Экземпляр ещё полностью функционален. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `onUnmounted()` | Регистрирует callback после размонтирования компонента. | — | <pre><code>import { onMounted, onUnmounted } from 'vue'<br><br>let intervalId<br>onMounted(() => { intervalId = setInterval(() => {}) })<br>onUnmounted(() => clearInterval(intervalId))</code></pre> | Composition API. Все дочерние компоненты размонтированы, reactive effects остановлены. Для очистки таймеров, listeners, соединений. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `onErrorCaptured()` | Регистрирует хук при перехвате ошибки от descendant-компонента. | — | <pre><code>import { onErrorCaptured } from 'vue'<br><br>onErrorCaptured((err, instance, info) => {<br>  console.error(err, info)<br>  return false // остановить propagation<br>})</code></pre> | Composition API. Аргументы: err, instance, info. Перехватывает ошибки из render, handlers, lifecycle, setup, watchers, directives, transitions. return false останавливает propagation. В production info — сокращённый код. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle), [Production Error Code Reference](https://vuejs.org/error-reference/) |
| `onRenderTracked()` | Debug-хук при отслеживании reactive dependency render effect. | — | <pre><code>import { onRenderTracked } from 'vue'<br><br>onRenderTracked((e) => {<br>  console.log(e.key, e.type, e.target)<br>})</code></pre> | Composition API. Только dev-режим. Не вызывается при SSR. DebuggerEvent: effect, target, type ('get'/'has'/'iterate'), key. | [Reactivity in Depth](https://vuejs.org/guide/extras/reactivity-in-depth) |
| `onRenderTriggered()` | Debug-хук при повторном запуске render effect из-за dependency. | — | <pre><code>import { onRenderTriggered } from 'vue'<br><br>onRenderTriggered((e) => {<br>  console.log(e.key, e.type, e.newValue)<br>})</code></pre> | Composition API. Только dev-режим. Не вызывается при SSR. DebuggerEvent: type ('set'/'add'/'delete'/'clear'), key, newValue, oldValue. | [Reactivity in Depth](https://vuejs.org/guide/extras/reactivity-in-depth) |
| `onActivated()` | Callback после вставки экземпляра в DOM из кэша KeepAlive. | — | <pre><code>import { onActivated } from 'vue'<br><br>onActivated(() => {<br>  // компонент снова активен<br>})</code></pre> | Composition API. Не вызывается при SSR. | [Lifecycle of Cached Instance](https://vuejs.org/guide/built-ins/keep-alive#lifecycle-of-cached-instance) |
| `onDeactivated()` | Callback после удаления экземпляра из DOM в кэш KeepAlive. | — | <pre><code>import { onDeactivated } from 'vue'<br><br>onDeactivated(() => {<br>  // компонент деактивирован, но не уничтожен<br>})</code></pre> | Composition API. Не вызывается при SSR. | [Lifecycle of Cached Instance](https://vuejs.org/guide/built-ins/keep-alive#lifecycle-of-cached-instance) |
| `onServerPrefetch()` | Регистрирует async-функцию, выполняемую до SSR-рендера компонента. | — | <pre><code>import { ref, onServerPrefetch, onMounted } from 'vue'<br><br>const data = ref(null)<br>onServerPrefetch(async () => {<br>  data.value = await fetchOnServer()<br>})<br>onMounted(async () => {<br>  if (!data.value) data.value = await fetchOnClient()<br>})</code></pre> | Composition API. Если возвращает Promise, сервер ждёт resolve. Только SSR. Для prefetch данных на сервере. | [Server-Side Rendering](https://vuejs.org/guide/scaling-up/ssr) |
| `beforeCreate` | Вызывается при инициализации экземпляра. | — | <pre><code>export default {<br>  beforeCreate() {<br>    // props resolved, state ещё настраивается<br>  }<br>}</code></pre> | Options API. Сразу после инициализации, props определены как reactive. setup() Composition API вызывается раньше beforeCreate. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `created` | Вызывается после настройки всего state-related options. | — | <pre><code>export default {<br>  created() {<br>    // data, computed, methods, watchers готовы<br>    // $el ещё недоступен<br>  }<br>}</code></pre> | Options API. Reactive data, computed, methods, watchers настроены. Фаза монтирования ещё не началась. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `beforeMount` | Вызывается непосредственно перед монтированием компонента. | — | <pre><code>export default {<br>  beforeMount() {<br>    // DOM ещё не создан<br>  }<br>}</code></pre> | Options API. Reactive state готов, DOM-узлы ещё не созданы. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `mounted` | Вызывается после монтирования компонента. | — | <pre><code>export default {<br>  mounted() {<br>    console.log(this.$el)<br>  }<br>}</code></pre> | Options API. DOM создан и вставлен. Для side effects с доступом к DOM. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `beforeUpdate` | Вызывается перед обновлением DOM из-за reactive state change. | — | <pre><code>export default {<br>  beforeUpdate() {<br>    // DOM до обновления<br>  }<br>}</code></pre> | Options API. Можно безопасно изменять state. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `updated` | Вызывается после обновления DOM. | — | <pre><code>export default {<br>  updated() {<br>    // DOM обновлён<br>  }<br>}</code></pre> | Options API. Не мутировать state в updated. Для конкретного DOM после изменения — nextTick(). Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `beforeUnmount` | Вызывается непосредственно перед размонтированием. | — | <pre><code>export default {<br>  beforeUnmount() {<br>    // экземпляр ещё функционален<br>  }<br>}</code></pre> | Options API. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `unmounted` | Вызывается после размонтирования компонента. | — | <pre><code>export default {<br>  unmounted() {<br>    clearInterval(this.intervalId)<br>  }<br>}</code></pre> | Options API. Для очистки side effects. Не вызывается при SSR. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle) |
| `errorCaptured` | Вызывается при перехвате ошибки от descendant-компонента. | — | <pre><code>export default {<br>  errorCaptured(err, instance, info) {<br>    console.error(err, info)<br>    return false<br>  }<br>}</code></pre> | Options API. return false останавливает propagation. Осторожно с async setup() и Suspense — возможны ошибки и из setup, и из render. | [Lifecycle Hooks](https://vuejs.org/guide/essentials/lifecycle), [Production Error Code Reference](https://vuejs.org/error-reference/) |
| `renderTracked` | Debug-хук при tracking dependency render effect. | — | <pre><code>export default {<br>  renderTracked(e) {<br>    console.log(e.key, e.type)<br>  }<br>}</code></pre> | Options API. Только dev. Не вызывается при SSR. | [Reactivity in Depth](https://vuejs.org/guide/extras/reactivity-in-depth) |
| `renderTriggered` | Debug-хук при re-run render effect. | — | <pre><code>export default {<br>  renderTriggered(e) {<br>    console.log(e.key, e.type)<br>  }<br>}</code></pre> | Options API. Только dev. Не вызывается при SSR. | [Reactivity in Depth](https://vuejs.org/guide/extras/reactivity-in-depth) |
| `activated` | Вызывается после вставки экземпляра из кэша KeepAlive. | — | <pre><code>export default {<br>  activated() {<br>    // компонент снова активен<br>  }<br>}</code></pre> | Options API. Не вызывается при SSR. | [Lifecycle of Cached Instance](https://vuejs.org/guide/built-ins/keep-alive#lifecycle-of-cached-instance) |
| `deactivated` | Вызывается после удаления экземпляра в кэш KeepAlive. | — | <pre><code>export default {<br>  deactivated() {<br>    // компонент деактивирован<br>  }<br>}</code></pre> | Options API. Не вызывается при SSR. | [Lifecycle of Cached Instance](https://vuejs.org/guide/built-ins/keep-alive#lifecycle-of-cached-instance) |
| `serverPrefetch` | Async-функция, выполняемая до SSR-рендера компонента. | — | <pre><code>export default {<br>  data() { return { data: null } },<br>  async serverPrefetch() {<br>    this.data = await fetchOnServer()<br>  }<br>}</code></pre> | Options API. Promise блокирует SSR-рендер до resolve. Только SSR. | [Server-Side Rendering](https://vuejs.org/guide/scaling-up/ssr) |
